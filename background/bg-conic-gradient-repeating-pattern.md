## One line of code repeats the angular gradient code to achieve cool patterns

One line of code repeats the angular gradient code to achieve cool patterns.

Specifically, you can read this article of mine:

SCSS：
```scss
html {
    height: 100%;
    background: repeating-conic-gradient(#fff, #000, #fff 0.1deg);
}
```
